  2. 枠線を消す/つける
    3. 「枠線を消す/つける」という編集ボタンを押して変えます。
    なお、以下説明することは、基本的に枠線を出してからして下さい。
  3. 編集の仕方
    4. まず、枠線を出します。
    次に、編集したいセルをクリックして、セルの枠の色を変えます。
    最後に、素材テーブルの好きなところをクリックすると、セルにその素材が入ります。
  4. 行数・列数の増やし方/減らし方
    5. まず、枠線を出します。
    次に、それぞれ対応するボタンを押すと操作が行えます。
  5. 出力の仕方
    6. 「出力する」という編集ボタンを押すと操作が行えます。
    なお、この機能のみ枠線を出す必要はありません。
  6. 入力の仕方
    7. まず、枠線を出します。
    データBOXにデータを入れて、「入力する」という編集ボタンを押すと操作が行えます。
  7. 回路図をパソコン上に保存する方法
    8. 枠線を消し、回路テーブルをドラッグ、右クリックして「コピー」を選びます。
    次に、Wordなどに「貼り付け」すると入る・・・かも。
